Project 1: Pixel Art Maker

HTML5, CSS3, jQuery

About the Project

This is a simple desktop game that was my final project for Google African Challenge Scholarship 3.0 with Andela, by Udacity,USA.

It allows the player to enter the desired dimensions of the grid they would like created for them on the desktop. This is done by clicking on the number input until the required value of the width and height are set. For this case, the maximum dimension has been set to 50 by 50. They can the click on the submit button to enable creation of the grid with the specified dimensions. They can proceed to select a colour of their choice and apply to any box of their choice within the created grid.Only a single colour can be picked and applied at a time. Enjoy!
